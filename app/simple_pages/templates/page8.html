class Calculator:
""" This is the default result property"""
result = 0
def add(self, value_1):
""" This is the add method"""
self.result = self.result + value_1
return self.result
def subtract(self, value_1):
""" This is the subtract method"""
self.result = self.result - value_1
return self.result
def get_result(self):
""" This is the get result method"""
return self.result

Single-Responsibility Principle

being able to split the operations of the code into subparts and making a single function class, main. The main class will call all the other functions one-by-one during a step by step process.

The Open-Closed Principle

Creating new method functions should add an extension to the main class.
